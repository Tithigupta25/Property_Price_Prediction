# Property_Price_Prediction

This project predicts property prices using machine learning.  
It is built using FastAPI and a trained model to give quick predictions based on property features.

# Tech Used
Python
FastAPI
Scikit-learn
Pandas

# Dataset
The dataset includes property details like area, bedrooms, location, and other features.  
It is available in the `data` folder.

# How to Run

Terminal-----

git clone https://github.com/Tithigupta25/Property_Price_Prediction.git
cd Property_Price_Prediction
pip install -r requirements.txt
uvicorn app:app --reload
